# NepalTourAI

An Intelligent Personalized Tourism Recommendation and Itinerary Planning System for Nepal

## Project Overview

NepalTourAI is a full-stack web application that uses machine learning to provide personalized travel recommendations for tourists visiting Nepal. The system combines K-Nearest Neighbors (KNN) algorithm with Content-Based Filtering to match travelers with destinations that suit their preferences, budget, fitness level, and interests.

## Features

- **Personalized Recommendations**: ML-powered suggestions based on user preferences
- **Interactive Map**: Explore Nepal destinations with Leaflet.js and OpenStreetMap
- **Automatic Itinerary Generator**: Create day-by-day travel plans
- **User Preference Wizard**: Smart questionnaire to understand traveler needs
- **PDF Export**: Download your itinerary for offline use
- **Admin Dashboard**: Manage destinations and view analytics

## Technology Stack

### Frontend
- React.js 18+
- Tailwind CSS
- Leaflet.js (Interactive Maps)
- Axios (API calls)

### Backend
- Python FastAPI
- PostgreSQL Database
- scikit-learn (ML)
- pandas, NumPy
- JWT Authentication

### Machine Learning
- K-Nearest Neighbors (KNN)
- Content-Based Filtering (TF-IDF)
- Hybrid Recommendation System
